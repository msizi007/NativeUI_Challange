React Navigation v7: Static API

Based on the syntax you provided, you are using the Static API introduced in React Navigation v7. This is a modern, simplified way to define navigation.

Since you have the stack definition, you need two additional things to make it work:

  1. The actual Screen Component (HomeScreen).
  2. A way to render the navigation (createStaticNavigation).

Complete Implementation

Copy the code below into your main App.js or App.tsx file.

import React from 'react';
import { View, Text, StyleSheet } from 'react-native';
import { createStaticNavigation } from '@react-navigation/native';
import { createNativeStackNavigator } from '@react-navigation/native-stack';

// 1. Define your Screen Component
function HomeScreen() {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Welcome to Home!</Text>
    </View>
  );
}

// 2. Define your Stack (Your provided code)
const MyStack = createNativeStackNavigator({
  screens: {
    Home: {
      screen: HomeScreen,
      options: {
        title: 'My home',
        headerStyle: {
          backgroundColor: '#f4511e',
        },
        headerTintColor: '#fff',
        headerTitleStyle: {
          fontWeight: 'bold',
        },
      },
    },
  },
});

// 3. Create the Navigation Container
const Navigation = createStaticNavigation(MyStack);

// 4. Export the App
export default function App() {
  return <Navigation />;
}

// Simple styles for the demo
const styles = StyleSheet.create({
  container: {
    flex: 1,
    alignItems: 'center',
    justifyContent: 'center',
  },
  text: {
    fontSize: 20,
    color: '#333',
  },
});

Key Steps Explained

1. HomeScreen

You referenced screen: HomeScreen in your configuration, so we must define this function so React knows what visual elements to display.

2. createStaticNavigation

This is the "magic glue" in React Navigation v7. It takes your configuration object (MyStack) and automatically wraps it in a Navigation Container, handling all the state management for you.

3. App Component

Finally, we export an App component that simply returns the <Navigation /> component we created.

Required Dependencies

If you haven't installed the packages yet, run this in your terminal:

npm install @react-navigation/native @react-navigation/native-stack react-native-screens react-native-safe-area-context
Note on "One Page":
Using a Stack Navigator for just one page is perfectly fine! It provides the Header bar (styled in your signature orange color) and prepares your app to easily navigate to a second screen in the future.
